## 2. 診断ロジックの全貌（5問とタグ設定）

### ▼ Q1. 現在のあなたの役職は？
- 【選択肢1】代表取締役 / 経営者（アクション：タグ「役職_経営者」を追加 → Q2へ）
- 【選択肢2】役員 / マネージャー（アクション：タグ「役職_幹部」を追加 → Q2へ）
- 【選択肢3】店長 / 現場責任者（アクション：タグ「役職_店長」を追加 → Q2へ）

### ▼ Q2. 現在の店舗数・組織規模は？
- 【選択肢1】1店舗（アクション：タグ「規模_1店舗」を追加 → Q3へ）
- 【選択肢2】2〜5店舗（アクション：タグ「規模_2_5店舗」を追加 → Q3へ）
- 【選択肢3】6店舗以上（アクション：タグ「規模_6店舗以上」を追加 → Q3へ）

### ▼ Q3. 今、最も頭を抱えている「痛み」は？
- 【選択肢1】自分が現場を離れると質が落ちる（アクション：タグ「悩み_自分が現場」を追加 → Q4へ）
- 【選択肢2】採用してもすぐ辞める・育たない（アクション：タグ「悩み_採用定着」を追加 → Q4へ）
- 【選択肢3】売上や店舗展開が頭打ちしている（アクション：タグ「悩み_展開頭打ち」を追加 → Q4へ）

### ▼ Q4. 業務マニュアルは機能していますか？
- 【選択肢1】ない / 社長の頭の中にあるだけ（アクション：タグ「マニュアル_なし」を追加 → Q5へ）
- 【選択肢2】あるが形骸化（誰も読んでいない）（アクション：タグ「マニュアル_形骸化」を追加 → Q5へ）
- 【選択肢3】チェックリストとして機能中（アクション：タグ「マニュアル_機能中」を追加 → Q5へ）

### ▼ Q5. 1年後、会社をどうしたいですか？
- 【選択肢1】社長が現場を離れ、完全に自走させたい（アクション：タグ「目標_自走化」を追加 → 結果判定へ）
- 【選択肢2】理念に共感して長く働く右腕を育てたい（アクション：タグ「目標_右腕育成」を追加 → 結果判定へ）
- 【選択肢3】多店舗展開を加速させステージを上げたい（アクション：タグ「目標_規模拡大」を追加 → 結果判定へ）
